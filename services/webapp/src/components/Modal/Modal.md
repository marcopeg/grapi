
#### Fade

```js
initialState = { isVisible: false };
<div>
    <button onClick={() => setState({ isVisible: !state.isVisible })}>
        Toggle Modal
        {state.isVisible ? ' - visible' : null}
    </button>
    <Modal
        isVisible={state.isVisible}
        useBackdrop={true}
        onRequestHide={() => setState({ isVisible: !state.isVisible })}
    >
        Hello World - Fade!
    </Modal>
</div>
```
